## Smart Access Tracker - Android App

### Description
This app displays student attendance data in real-time using Firebase Firestore.

### Features
- View daily and monthly attendance records
- Real-time data sync with Firebase
- Login and registration system
- User-friendly interface

### APK Download
https://drive.google.com/file/d/1dwzBjPszTJ7o5tdSanPUl8I13bLr5QGF/view?usp=drive_link

### Activities in this app
- LoginActivity – Handles user login
- RegisterActivity – New user registration
- DashboardActivity – Main attendance overview
- DailyReportActivity – Day-wise attendance
- MonthlyReportActivity – Month-wise attendance
- IntroductionActivity – App intro/splash screen

### Setup
1. Download and install the APK
2. Connect to internet
3. Login and view attendance data
